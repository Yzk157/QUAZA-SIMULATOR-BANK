Simulador bancário desenvolvido para praticar lógica de programação, regras de negócio e manipulação de interface com JavaScript puro.

## Objetivo

Construir um sistema simples de operações financeiras para treinar conceitos fundamentais de desenvolvimento front-end e lógica aplicada.

## Funcionalidades

- Criação e gestão de saldo
- Depósito e saque com validações
- Fluxo de operações com feedback visual
- Atualização dinâmica da interface

## Tecnologias

- HTML5
- JavaScript (ES6+)
- CSS3 (se aplicável)
